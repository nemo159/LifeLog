# LifeLog 프로젝트 개요

**프로젝트 명**: LifeLog  
**상태**: Active Development (기능 확장 단계)  
**작성일**: 2026-01-10

## 프로젝트 비전
LifeLog는 기분, 메모, 사진을 통해 일상을 기록하는 개인용 저널링 애플리케이션입니다. 
사용자의 프라이버시를 최우선으로 하며, 모든 데이터는 서버가 아닌 **기기 내부에만 저장**됩니다.

## 핵심 기능 (Core Features)

1. **타임라인 뷰 (Timeline View)**
   - 날짜순(최신순/과거순) 정렬 기능을 제공합니다.
   - 년/월/일 단위로 기록을 그룹화하여 볼 수 있습니다.
   - 각 항목은 기분(이모지/아이콘)과 메모, 대표 사진을 미리 보여줍니다.

2. **기록 작성기 (Log Editor)**
   - 날짜 및 시간 선택.
   - 1~5점 척도의 기분 선택 및 **추가 태그/이벤트** 지정.
   - 텍스트 메모 작성.
   - **사진 첨부** (Android Photo Picker 사용, 내부 저장소 복사).
   - **위치 및 날씨** 자동 설정 (권한 허용 시).

3. **조회 및 관리**
   - **상세 화면**: 기록의 전체 내용과 고화질 사진을 확인합니다.
   - **삭제 기능**: 잘못 작성된 기록을 영구적으로 삭제합니다.
   - **캘린더 뷰**: 달력 형태로 기록 작성 여부와 기분 분포를 확인합니다.
   - **통계**: 월별/기간별 기분 변화 추이를 그래프로 제공합니다.

4. **시스템 및 설정**
   - **테마 변경**: 다크/라이트 모드 및 앱 포인트 컬러 변경 기능.
   - **백업/복구**: 구글 드라이브(Google Drive) 연동을 통한 데이터 안전 보관.
   - **로컬 저장소**: Room Database 및 파일 시스템을 이용한 안전한 데이터 관리.

## 기술 스펙 (Technical Specification)

### 기술 스택 (Tech Stack)
- **플랫폼**: Android (Target SDK 36+)
- **언어**: Kotlin
- **UI 프레임워크**: Jetpack Compose
- **의존성 주입 (DI)**: Hilt
- **데이터베이스**: Room
- **비동기 처리**: Coroutines & Flow
- **백그라운드 작업**: WorkManager
- **네트워크**: Retrofit/OkHttp (날씨 API, Google Drive API 용)

### 아키텍처 (Architecture)
**Clean Architecture** (Core, Data, Feature, Navigation) 구조를 유지합니다.
